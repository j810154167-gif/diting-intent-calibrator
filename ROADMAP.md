# Roadmap

## Near term

- Stabilize the Runtime Gate test matrix.
- Add more regression cases for multi-Agent handoff and nested boundary conflicts.
- Package the Skill for distribution after metadata validation.
- Add examples for common forwarding, local execution, and memory-candidate workflows.

## Mid term

- Add a structured JSON CLI output mode.
- Add configurable policy thresholds for density and leakage risk.
- Add failure-case to regression-ledger conversion helpers.
- Add optional integration points for external Agent handoff controllers.

## Long term

- Upgrade from local Runtime Gate prototype to AgentOS-style runtime middleware.
- Add durable memory store integration with explicit human approval.
- Add observability for gate decisions, stop reasons, and human confirmations.
- Support multi-target Agent routing with per-agent boundary contracts.
